## 🔗 How Files Link Together

### From index.html → Project Pages
```html
<!-- In index.html, projects section -->
<a href="projects/alkaline-electrolyzer.html">View Details</a>
```

### From Project Pages → Back to index.html
```html
<!-- In projects/alkaline-electrolyzer.html -->
<a href="../index.html">Back to Portfolio</a>
<a href="../index.html#projects">Projects</a>
<a href="../index.html#contact">Contact</a>
```

## 🚀 Simple Upload Method (Recommended for Beginners)

### Method 1: Direct GitHub Upload (No Git Knowledge Required)

1. **Create Repository:**
   - Go to github.com
   - Click "+" → "New repository"
   - Name: `portfolio` or `your-name-portfolio`
   - Public repository
   - Click "Create"

2. **Upload Files:**
   - Click "Add file" → "Upload files"
   - Drag your `index.html` into the upload area
   - Create commit message: "Add homepage"
   - Click "Commit changes"

3. **Create Projects Folder:**
   - Click "Add file" → "Create new file"
   - Type: `projects/alkaline-electrolyzer.html`
   - Paste your project page HTML
   - Commit changes

4. **Enable GitHub Pages:**
   - Go to Settings
   - Click "Pages" in sidebar
   - Source: Deploy from main branch
   - Root folder: `/`
   - Save

5. **Access Your Site:**
   - URL: `https://your-username.github.io/portfolio/`
   - Wait 1-2 minutes for deployment

## 💡 Pro Tips

1. **Use lowercase and hyphens** for file names
   - ✅ Good: `alkaline-electrolyzer.html`
   - ❌ Bad: `Alkaline Electrolyzer.html`

2. **Keep folder structure simple** at first
   - Start with just `index.html` and `projects/`
   - Add `assets/` later when you have images

3. **Test every link** before uploading
   - Broken links are the #1 issue

4. **Use relative paths**, not absolute
   - ✅ `projects/page.html`
   - ❌ `/projects/page.html`

5. **Commit often** with clear messages
   - "Add project detail page"
   - "Update contact information"
   - "Fix broken links"

## 🆘 Common Problems & Solutions

**Problem:** "My project page shows 404 error"
**Solution:** Check the link path is correct - should be `projects/alkaline-electrolyzer.html`

**Problem:** "Back button doesn't work on project page"
**Solution:** Use `../index.html` not `index.html` from project folder

**Problem:** "CSS isn't loading"
**Solution:** Font Awesome CDN must be in `<head>` of every page

**Problem:** "Images don't show"
**Solution:** Check image paths - use `../assets/images/` from project pages

**Problem:** "GitHub Pages shows README instead of website"
**Solution:** Make sure your main file is named `index.html` (lowercase)
